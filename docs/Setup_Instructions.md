# Setup Instructions – LearnTrack Project

## JDK Installation
For this project, I used:

- JDK Version: Java 17 (or Java 11 / Java 8 – update as applicable)
- IDE: IntelliJ IDEA

Java was installed from the official Oracle / OpenJDK website and added to the system PATH.

## Verifying Java Installation
To verify Java installation, the following commands were run in terminal:

```bash
java -version
javac -version
